WebJack-Firmata Demo

WebJack is a JavaScript library that uses an audio modem to communicate with an Arduino ÂµC via headphone jack. Firmata is a RPC like protocol, that defines a serial interface to read from or write to a microcontroller's pins. This demo combines WebJack and Firmata to control an Arduino via headphone jack.

To run the demo, upload this modified version of the StandardFirmata sketch.
